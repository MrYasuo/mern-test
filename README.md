## Installation

```bash
git clone https://github.com/MrYasuo/mern-test.git
cd mern-test
npm i
cd client && npm i
```

## Usage

1. Configure your **environment** variables
   ```bash
   MONGODB_URI=your_mongodb_uri
   JWT_SECRET=your_jwt_secret
   ```
2. Run the server
   ```bash
   cd mern-test
   npm run dev
   ```
